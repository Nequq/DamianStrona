Kaskadowe arkusze stylów to zbiór reguł, które opisują, jak dany element strony ma wyglądać.
Reguły CSS tworzą hierarchię, a ich kolejność jest określona przez kaskadę.
To znaczy, że jeśli dwie reguły mają takie same selektory, to ta, która występuje później, ma pierwszeństwo.



Cascading style sheets are a set of rules that describe what a given page element should look like.
CSS rules form a hierarchy and their order is determined by a cascade.
This means that if two rules have the same selectors, the one that comes later takes precedence.
